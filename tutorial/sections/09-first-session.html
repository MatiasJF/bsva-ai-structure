Your first real session.

Pick a real task (1–2 hours, not Restricted). Copy these prompts. Watch the cycle pay off.

Step 1 — Start Claude in the project folder

shell cd ~/projects/my-new-task claude

Step 2 — Open with a grounding prompt

Describe the desire, don't jump to "write me X".

prompt I want to [your task, one sentence]. Before we start, walk me through a CLARIFY round — what do you need from me?

Step 3 — Let Claude propose the Nestr capture

prompt Good. Now propose the Nestr node — nest or tension? Show me the title, acceptance criteria, and labels before creating.

Step 4 — Plan-mode

prompt Enter plan-mode and draft the execution plan. Reference the nest ID.

Or press shift + tab to toggle plan-mode directly.

Step 5–6 — Discuss + approve

Read the plan. Push back on anything. Iterate. When ready:

prompt Approved. Go.

Step 7–8 — Claude scaffolds + executes

Milestone comments post to the Nestr nest. Scope changes pause and re-approve.

Step 9 — Report on close

prompt Work is done. Write the REPORT comment on the Nestr nest — outputs, plan deltas, any new tensions spawned, promotable learnings.

Step 10 — You close the loop

You (not Claude) review acceptance criteria. If met, set project.status: Done.

What you should feel afterwards
  • Mild friction in CLARIFY — gone by week two.
  • Surprise that plan-mode caught a scope issue.
  • A well-structured Nestr nest you can re-use as a template.
  • A prompt you re-typed — a candidate for a skill.
If something slipped
