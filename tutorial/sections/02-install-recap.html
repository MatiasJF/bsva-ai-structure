Three things to do now

1. Wire up your Nestr MCP

Open ~/.claude/mcp.bsva-template.json, replace <your-nestr-api-key>, merge the mcpServers block into ~/.claude/mcp.json. Restart Claude.

2. Review the settings template

Peek at ~/.claude/settings.bsva-template.json. Copy the deny rules across. Don't relax them.

3. Verify

Ask Claude in any session:

prompt Use the Nestr MCP to tell me who I am and which workspace I am in.

If you see your name + workspace, you're wired up. Auth error → revisit step 1.

Re-running later

Run ./install.sh or ./install.ps1 again anytime — idempotent. Add --sync after git pull to update global pieces without the department prompt.
