# bsva-onepager

Pointer skill — see `../bsva-docx/SKILL.md` for the pattern.

## When Claude should invoke this

- User asks for a one-pager / fact sheet / briefing handout
- User wants to turn MD content into a BSVA-branded PDF
- Marketing / biz-dev content targeted at partners or policymakers

## Classification rules

One-pagers are often used externally. Default classification is **Public** — but if the content references unannounced partners, embargoed news, or financials, escalate tier to at least **Internal** and confirm with the user.
