1. `<repo>/workflow/README.md` and the numbered files there.
2. `typical-artifacts.md` (this folder) — what DevRel actually ships.
3. `clarify-questions.md` (this folder) — the DevRel-specific CLARIFY bank.
4. `nest-templates.md` and `tension-templates.md` — canonical Nestr shapes.
5. `scaffold-matrix.md` — which template + skills + MCPs per artifact.
