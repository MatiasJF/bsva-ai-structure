# new-sdk-example

A minimal starter for a BSV SDK example project.

## What this produces

A runnable, minimal example that illustrates **one concept** from the `@bsv/sdk`. Targets TypeScript / Node.js.

## How to use

```bash
cp -R ../new-sdk-example ~/projects/my-example
cd ~/projects/my-example
git init && git add . && git commit -m "scaffold from new-sdk-example"
ln -s ~/bsva-ai-structure/departments/developer-relations/CLAUDE.md CLAUDE.md
npm install
npm start
```
