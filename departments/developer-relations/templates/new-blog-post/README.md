# new-blog-post

Template for a BSVA blog post. Markdown-only, ready to drop into BSVA's blog repo.

## Usage

```bash
cp -R ~/bsva-ai-structure/departments/developer-relations/templates/new-blog-post \
      ~/posts/my-post
cd ~/posts/my-post
```

Edit `post.md`. When ready, move it to the BSVA blog repo and PR.

## Frontmatter conventions

See `post.md` — BSVA blog expects specific fields.

## Voice

See `guides/for-claude/brand-voice.md`.

## Check before publishing

- [ ] Lead with the question or claim.
- [ ] Paragraph 1 says what the reader will understand by the end.
- [ ] Body follows mechanism → example → implication.
- [ ] Close links the reader to relevant spec / repo / tutorial.
- [ ] No hype, no price talk, no "revolutionary".
- [ ] Code snippets run.
- [ ] Images have alt text.
