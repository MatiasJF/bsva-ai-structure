# Technology templates

Starter scaffolds for Tech / Engineering projects.

| Template | For |
|---|---|
| `new-service/` | A backend service (HTTP, worker, etc.). |
| `new-protocol-spec/` | A proposed BSVA protocol document / BRC-style spec. |
| `new-research-note/` | Internal research note or analysis document. |

See `guides/visuals/06-project-scaffolding.md` for the usage flow.
