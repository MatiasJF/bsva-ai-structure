# Executive — brief builder playbook

Executive digests from public sources. Used for: pre-meeting briefings, industry scans, policy updates.

## In scope

- Industry reports (freely available).
- Regulatory publications.
- Public filings of industry peers.
- Public BSVA statements and published industry commentary.

## Out of scope

- BSVA's internal positioning notes.
- Specific partner private posture.
- Anything that combines public data with BSVA's unreleased stance.

## The cycle

### 1. CLARIFY
- Topic.
- Time horizon.
- Depth (2-page summary / 8-page deep-dive / 1-page talking points).
- Audience within BSVA (exec-only / wider leadership / full org).
- Source restriction (public sources only — explicit confirmation).

### 2. CAPTURE
**Nest** under Exec role. Restricted session; artifact Internal (for circulation inside BSVA) or Confidential if the synthesis reveals strategic angles.

Title: `[Topic] exec brief shipped`

Acceptance criteria:
- Public sources only; each claim cited.
- Time horizon covered with dated entries.
- Signal vs noise separated — the 3–5 things that actually matter.
- BSVA-context commentary (if any) clearly separated from factual summary.

### 3. EXECUTE
Draft:
- Executive summary (3 bullets).
- Timeline of relevant public events.
- Key actors and their public positions.
- Emerging themes.
- (Optional) BSVA-context commentary — clearly marked, Internal-scoped.

### 4. REPORT
Comment on nest: brief ready, N citations, signal/noise split noted.

### 5. CLOSE
Exec reviews → circulates per scope.

## Hard rules

- **Never** combine public data with BSVA's private strategy in the same artifact. If context is needed, it goes in a separate Internal note that doesn't reference Claude-produced material directly.
- **Never** fabricate citations.
- **Never** access subscription databases without explicit ToS confirmation + BSVA contract coverage.
- **Never** produce predictions without grounding them in cited public signals.

## When the user wants Claude to "give their take"

Claude may offer analysis *grounded in the public sources shown*, but:
- Must separate analysis from factual summary.
- Must caveat uncertainty.
- Must not claim BSVA's position — that's Exec's to state.
