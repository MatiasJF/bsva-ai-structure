# Marketing — event playbook

From event announcement to follow-up.

## Phases

1. **Announce** — landing page + invite email + social launch.
2. **Build** — nurture sequence, speaker reveals, partner mentions (per Biz-Dev clearance).
3. **Deliver** — day-of social coverage, live updates.
4. **Follow-up** — thank-you emails, recording release, retro content.

Each phase = nest under Marketing role. Linked to a parent event-nest.

## CLARIFY for event marketing

- Event type (BSVA-hosted / co-hosted / sponsored / speaking-slot).
- Audience + attendance target.
- Venue (physical / virtual / hybrid) + time zone.
- Which partner names are public for this event? Which are not?
- Sponsorship asks ($ amounts) — `[AMOUNT]` placeholders only in any drafts.
- Exec involvement? Keynote? Panel?

## Partner coordination

For every event with partner involvement:
- Tension to Biz-Dev for name clearances (batched across the campaign).
- Any partner-authored content (logos, quotes, blurbs) must be partner-approved outside Claude; Claude may format but not originate.
- Embargoes: if a partner has pre-announcement embargo, Marketing stays silent until the embargo lifts; Claude respects the embargo as a hard constraint.

## Live-event operations

- No "generate live tweets on the fly via Claude" at events. Drafts prepared beforehand, scheduled or manually posted during.
- Any reactive posting (e.g., responding to a keynote) goes through `external-comm-check` before posting.
- Photographs / videos of attendees: consent before sharing externally; BSVA's event-consent process applies.

## Post-event

- Recording release: tension to Ops for upload scheduling; tension to Legal if any clips need review; `classify-before-publish` gate on the final.
- Thank-you emails: draft, `external-comm-check`, send from approved sending account.
- Retro nest: what went well, what to change, promotable patterns.

## Hard rules

- **Never** name an unannounced co-sponsor / partner.
- **Never** publish attendee names without consent.
- **Never** use AI-generated imagery of real attendees or speakers.
- **Never** speculate on undisclosed commercial details of sponsorships.
