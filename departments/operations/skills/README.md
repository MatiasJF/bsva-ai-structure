# Operations skills

Cross-Operations skills. Sub-sections (biz-dev, marketing, legal, exec) may have their own additional skills.

## Candidates

- `proposal-shell` — generate a partnership proposal structure with placeholders throughout.
- `anonymizer` — scan content for names / amounts / dates and replace with placeholders.
- `meeting-prep` — given an agenda + attendees (roles only), draft a brief.
- `external-comm-check` — draft of an outgoing message → review for tone, disclosure risk, attachment safety.

All Operations skills default to `classification: Confidential` and `owner: "@bsva/operations"`. Security review is required before merging to this folder.
