# Education department skills

Skills for Education / BSV Academy work.

## Candidates

- `lesson-outline-builder` — topic + audience → multi-section outline with objectives.
- `quiz-generator` — lesson content → 5–10 assessment items with rubric.
- `reading-level-checker` — prose → reading-level estimate + suggestions to simplify.
- `academy-term-linker` — prose → prose with inline links to BSV Academy concepts.
- `bilingual-side-by-side` — English source → 2-column EN/target layout ready for review.

See `guides/visuals/03-skill-lifecycle.md`.

## Conventions

- `classification: Internal` default; `Public` for skills producing published-tier output.
- `owner: "@bsva/education"`.
- Prefer calling the BSV Academy MCP over improvising definitions.
