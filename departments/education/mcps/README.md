# Education department MCPs

MCPs scoped to Education work. The global BSV Academy MCP covers most needs.

## Candidates

- **LMS MCP** — read-only scoped to BSVA's learning platform for pulling course state / learner cohorts (no PII exposure).
- **Translation memory MCP** — internal TMX / TM-backed translator that preserves BSVA's terminology choices across translations.

Additions follow `guides/visuals/04-mcp-lifecycle.md`. LMS-related MCPs will need Education + Security + Legal review because of learner-data implications.
